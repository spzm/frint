Component

Component

Component classes ready to be embedded as JSX for rendering.

Usage

/** @jsx h */
import { createComponent, h } from 'frint';

const MyComponent = createComponent({
  render() {
    return (
      <p>Name is {this.prop.name}</p>
    );
  }
});

It is best practice to define propTypes while defining the Component too. You can read more about it in PropTypes section.
